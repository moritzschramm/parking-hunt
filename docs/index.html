ParkingHunt Rest API v1 Documentation

Requests

Basic JSON request structure

{
  "Credentials": {
    "accID": "accID",
    "ApiKey": "ApiKey"
  },
  "param1": your_value,
  "param2": your_value
}
Path Type Description Credentials required
/api/v1 GET Get API version (test API call) no
/api/v1/login POST Check if password matches email and send accID and ApiKey back no
/api/v1/register POST Register a new user no
/api/v1/spot/{spotID} POST Get all information of a spot yes
/api/v1/spots POST Get all created spots yes
/api/v1/spot POST Create a new spot yes
/api/v1/spot/{spotID} PUT Edit a spot (only if you are the spot creator) yes
/api/v1/spot/{spotID} DELETE Delete a spot (only if you are the spot creator) yes
/api/v1/acc/{accID} POST Get your account information yes
/api/v1/acc/{accID} PUT Edit your account details yes
/api/v1/acc/{accID} DELETE Delete your account yes
/api/v1/search/geo POST Searches for Spots near given GeoLocation (Lat/Lng) yes
/api/v1/search/add POST Add a Search yes
